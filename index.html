DESENVOLVIMENTO DE SISTEMAS - PROGRAMAÇÃO WEB

Para o desenvolvimento desse projeto, utilizei recursos do html, css e bootstrap.

O HTML é o componente básico da web, na qual a linguagem é escrita nas páginas da Web e interpretada pelo navegador.

O CSS foi desenvolvido para habilitar a separação do conteúdo e formato de um documento, incluindo elementos como cores, formatos de fontes e layout..

O Bootstrap é uma ferramenta gratuita para desenvolvimento HTML, CSS e JS, que auxilia no desenvolviemnto de projetos responsivos para dispositivos móveis na web.
